# AskUI Shell API Documentation

## Integrating Python API Documentation

To integrate Python API documentation from pydoc into this documentation:

1. Generate API documentation using pydoc:
   ```bash
   pydoc -w your_module
   ```

2. The generated HTML files can be converted to MDX format and integrated into this documentation.

3. Alternatively, you can use tools like `pdoc3` or `sphinx` which provide more modern documentation generation capabilities and better integration options.

For detailed API documentation, we recommend using one of these approaches:

- **Direct pydoc integration**: Generate and convert pydoc output
- **Sphinx**: More feature-rich documentation with better theme support
- **pdoc3**: Modern Python API documentation generator

Please ensure to regenerate the documentation whenever the API changes to keep it up to date.
